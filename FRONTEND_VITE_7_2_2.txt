VITE_API_URL=http://localhost:8000/api
VITE_APP_NAME=CAPTAR v2.0

Como usar em código:
  const apiUrl = import.meta.env.VITE_API_URL
  const appName = import.meta.env.VITE_APP_NAME

Variáveis disponíveis:
  - import.meta.env.VITE_* (acessíveis no cliente)
  - import.meta.env.MODE (development/production)
  - import.meta.env.DEV (true em desenvolvimento)
  - import.meta.env.PROD (true em produção)

DEPOIS:

  ✅ Funciona corretamente

Mudança:
  - process.env (Node.js) → import.meta.env (Vite)
  - REACT_APP_* → VITE_*

                        TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════════

Erro: "Cannot find module"
  - Solucao: npm install
  - Ou: Feche e execute novamente

Erro: "VITE_API_URL is not defined"
  - Solucao: Certifique-se que .env existe
  - Ou: Reinicie o servidor

Erro: "Connection refused" (ao conectar ao backend)
  - Solucao: Certifique-se que backend esta rodando
  - Verifique: http://localhost:8000

Erro: "CORS error"
  - Solucao: Backend deve ter CORS habilitado
  - Verifique: main.py ou main_extended.py
