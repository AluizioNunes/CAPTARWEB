═══════════════════════════════════════════════════════════════════════════════
                        TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════════

Erro: "Port 3000 already in use"
  - Solucao: Mude a porta em vite.config.ts
  - Ou: Feche a aplicacao que esta usando porta 3000

Erro: "Cannot find module"
  - Solucao: npm install
  - Ou: Feche e execute novamente

Erro: "Connection refused" (ao conectar ao backend)
  - Solucao: Certifique-se que backend esta rodando
  - Verifique: src/services/api.ts

Erro: "CORS error"
  - Solucao: Backend deve ter CORS habilitado
  - Verifique: main.py ou main_extended.py
