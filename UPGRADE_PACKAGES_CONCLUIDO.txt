                        PROXIMAS ACOES
═══════════════════════════════════════════════════════════════════════════════

PROBLEMA: Espaço em disco insuficiente

SOLUCAO:

1. Liberar espaço em disco:
   - Deletar arquivos desnecessarios
   - Limpar cache do Windows
   - Usar disco externo

2. Apos liberar espaço:
   cd c:\www\Streamlit\Captar\CAPTAR\Frontend
   npm cache clean --force
   npm install

3. Iniciar servidor:
   npm run dev

4. Abrir navegador:
   http://localhost:3000/
